闭包与匿名函数非常相似，但功能更强大。它们之间的关键区别是匿名函数本身没有上下文，而闭包有。让我们重写前面的例子，添加一个任意的数，而不是1:

```go
func main(){
    addN := func (m int){
        return func (n int){
            return m + n
        }            
    }
    
    addFive := addN(5)
    result := addN(6)
    // 5 + 6 must print 7 
    
    println(result)
}
```

`addN` 变量指向一个返回另一个函数的函数。但是返回的函数有 `m` 参数的上下文。每次调用 `addN` 都会创建一个具有固定m值的新函数，所以我们有主要的 `addN` 函数每个次加不同的值。

闭包的这种能力对于创建库或处理不支持类型的函数非常有用。
